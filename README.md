# Bike Sales Data Analysis Dashboard

## Project Overview
This project analyzes bike sales data using Microsoft Excel. The goal was to clean raw data, perform exploratory analysis, and build an interactive dashboard to uncover customer purchasing patterns.

The analysis focuses on customer demographics, commute distance, income level, and age bracket to identify trends influencing bike purchases.

## Objectives
- Clean and preprocess raw sales data
- Create Pivot Tables for structured analysis
- Identify key factors affecting bike purchases
- Build an interactive Excel dashboard
- Extract actionable business insights

## Tools Used
- Microsoft Excel
  - Data Cleaning
  - Pivot Tables
  - Pivot Charts
  - Dashboard Design
  - Slicers for interactivity

## Data Cleaning Process
The following preprocessing steps were performed:

- Removed duplicate entries
- Standardized categorical values (e.g., marital status, education)
- Checked for missing values
- Created calculated fields where necessary
- Ensured consistent formatting for analysis

## Dashboard Features
The dashboard includes:

- Customer Commute Distance vs Purchase
- Average Income Per Purchase (by Gender)
- Customer Age Bracket Analysis
- Interactive filters:
  - Marital Status
  - Region
  - Education Level

## Key Insights

- Customers with shorter commute distances are more likely to purchase bikes.
- Middle-aged customers (31+) show the highest purchase rate.
- Higher income levels correlate with increased likelihood of purchasing.
- Male customers have a slightly higher average income among buyers.

## Business Recommendation

Based on the analysis, marketing efforts should focus on:
- Middle-aged individuals
- Higher-income segments
- Short-distance commuters

This could potentially improve conversion rates and maximize revenue.
